<h1 align="center"> Jogo de Adivinhação </h1>


## Bora jogar? 🎮

Este projeto acadêmico é um jogo de adivinhação interativo. O sistema gera aleatoriamente um número secreto e o desafio do usuário é tentar descobrir qual é, recebendo dicas do sistema se o número digitado é maior ou menor que o alvo.

---

## Lógica e Aprendizados

Para a construção do jogo, apliquei os conceitos de lógica abaixo:

*   **`while`**
*   **`if`, `else if`, `else`**
*   **`break`**
*   **`Math.random()` com `parseInt()`**
*   Para finalizar a mensagem exibida no alert, usei um **operador ternário** para definir se a palavra ficaria no singular ou plural.
